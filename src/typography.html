Typography

Two font families are available in the Template Package 4.x framework: Merriweather and Open Sans. These fonts were selected because they were designed for legibility in digital products and support complex content and scientific notation and content. Using a two font system also allows headings to be easily distinguished from regular text.

References: U.S. Web Design Standards, Google Fonts, Bootstrap 4 Typography

Using REMs

Headings and body text use a relative unit of measurement or REM. This allows a “base” text size to be specified and other elements to be relative to this value. For example, the Paragraph Text size is 17px – this is the base 1.0 rem. The H1 as 2.5rem would be 2.5 x 17px – 42.5px. This method helps ensure consistent sizing across browsers and allows adjustments for many elements to be made by changing a single value. The conversion from rems to pixels for web display are handled by browsers and each browser has its own rounding rules for pixels values that are not whole numbers.
