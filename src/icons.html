Icons

The Template Package 4.x framework includes icons to represent a variety of health topics and communication activities. The icon set also includes standard icons to represent links to social media sites and indicate links to files and external sites. All icons are managed as SVG files so they do not lose quality when changing size. SVGs also provide flexibility for color changes and allow for easier updates.

Usage

SVG icons can be placed on a page using several different methods, each of which have their own benefits.

Img Source

Using SVG icons as images is an option which generally works best when the SVG icon has the appropriate fill colors applied to it already.

SVG Source

When you want more control over SVG icons, and you're using them a limited number of times, you can embed the SVGs directly into the page.

SVG Symbol

Use SVG symbols when you want to use standard embedding, or if you're using the icons multiple times on the page. These icons can be different sizes, colors and have different styles.
