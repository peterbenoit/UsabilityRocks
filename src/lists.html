Lists

The Template Package 4.x framework includes styles for unordered and ordered lists with one or more columns. There are also additional style options. Any type of list can be used within regular page content or within Containers.

Lists can be displayed in single or multi-column layouts. The markup for multi-column lists can either be done using CSS list classes, or using Bootstrap columns. There are demos for each list type with different layouts.

List classes are named using the format .cc-{columns} for xs and .cc-{breakpoint}-{columns} for sm, md, lg, and xl.

Where columns is one of:
  • 1 - One Column
  • 2 - Two Columns
  • 3 - Three Columns
  • 4 - Four Columns
